## Publish Model
### GitHub Pages
Primary live host for now.

### itch.io
Recommended as the storefront, but requires the game page to exist in the itch account first.
After the page exists, a butler push can publish updates quickly.

## Safe Update Flow
1. change files locally
2. commit to `master`
3. push to GitHub
4. if needed, mirror deployable files to `gh-pages`
5. smoke test live URL on mobile and desktop
